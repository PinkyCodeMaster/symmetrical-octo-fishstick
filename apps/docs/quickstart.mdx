# Quickstart Guide

This guide will help you set up the Tasks Application development environment quickly.

## Prerequisites

- Node.js 18+ 
- pnpm
- Docker & Docker Compose
- Git

## Setup Steps

### 1. Clone the Repository

```bash
git clone <your-repo-url>
cd tasks-app
```

### 2. Install Dependencies

```bash
pnpm install
```

### 3. Environment Setup

Create `.env` files in each app directory:

```bash
# apps/server/.env
DATABASE_URL="postgresql://user:pass@localhost:5432/tasks"
JWT_SECRET="your-secret"
```

### 4. Start Development Servers

```bash
# Start all services
pnpm dev

# Or start specific services
pnpm dev:server    # Backend API
pnpm dev:web       # Web App
pnpm dev:admin     # Admin Dashboard
```

### 5. Access the Applications

- Backend API: http://localhost:9000
- Web App: http://localhost:3000
- Admin Dashboard: http://localhost:3001
- API Documentation: http://localhost:9000/doc
